এই playlist-এ আমরা Git এবং GitHub নিয়ে ধীরে ধীরে শিখব — একদম basic থেকে শুরু করে practical level পর্যন্ত।

প্রথমে বুঝবো version control কী এবং কেন এটা দরকার। তারপর Git install, configuration, repository তৈরি — এগুলো হাতে-কলমে করব। ধাপে ধাপে দেখবো daily workflow, branching, merging, pull request, আর team এ কিভাবে কাজ করতে হয়।

শেষের দিকে real project এ Git কিভাবে ব্যবহার হয় সেটাও দেখব।
