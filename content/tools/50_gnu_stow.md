GNU Stow is a symlink manager that helps you organize dotfiles in a clean, modular way.

Instead of copying configs, it creates symlinks from a central dotfiles repo into your home directory.

_Why use it ?_

- Clean dotfile structure
- Easy rollback
- Modular config per tool
